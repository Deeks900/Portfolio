OpenMate – Mentor-Mentee Management System

OpenMate is a full-stack platform connecting mentors and mentees. Users can sign up as mentors or mentees, manage their profiles, and get email notifications with links to connect for sessions. The platform handles media uploads via Cloudinary, stores data in PostgreSQL, and uses Angular for the frontend and Python for backend services.
